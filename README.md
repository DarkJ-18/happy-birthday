# Happy Birthday

Tarjeta de cumpleaños interactiva hecha con HTML, CSS y JavaScript.

## Personalizacion rapida

Puedes personalizar desde la URL usando estos parametros:

- name: nombre de la persona
- from: firma o dedicatoria
- theme: rosa (default), azul o sol

Ejemplo:

<a href="https://programmergaurav.github.io/happy-birthday/?name=Valeria&from=Tu+familia&theme=azul" target="blank">Abrir ejemplo personalizado</a>
